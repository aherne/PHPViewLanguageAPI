escape

Marks tag body to be ignored by ViewLanguage compiler.

Syntax:

<escape> ... </escape>

Examples how this tag is compiled into PHP:

<escape> ${foo.bar} </escape>  
${foo.bar}
 

import

Includes another view language template into current one.

Syntax:

<import file="..."/>

Attributes:

NAME

DESCRIPTION

MANDATORY

DATA TYPE

file

Name of file whose sources should replace tag declaration. File will be located using extension & template folder supplied to compiler. When sources are parsed, same operation repeats recursively until no import tags are left.

Y

STRING

For examples how this tag is compiled into PHP, have a look here!

namespace

Marks custom location of user defined tag library.

Syntax:

<namespace taglib="..." folder="..."/>

Attributes:

NAME

DESCRIPTION

MANDATORY

DATA TYPE

taglib

Name of tag library to look for.

Y

STRING

folder

Location on disk (absolute or relative to project) where to look for tags (files) in library.

Y

STRING
